Instalación de CodeBlocks & creación del primer proyecto.

Agosto, 29, 2019 Chunche95

Para la instalación de CodeBlocks debemos ir a la página oficial del IDE:

Página codeblocks.
Otra opción es realizar la instalación mediante la línea de comandos, en el caso de Linux, por ejemplo es:
sudo apt-get install codeblocks -y

Creación de primer proyecto.

Una vez instalado el IDE, podemos crear un nuevo proyecto, para ello vamos al botón de "Create a new porject". Seguidamente seleccionaremos en "template" > Empty project > Next > Nombre del proyecto - "Aprendiendo C"; seguido de la ubicación donde guardar el proyecto /home/chunche/AprendiendoC/> Next > Seleccionamos el compilador a usar. > Finish.
Vamos al botón de la barra superior y Seleccionamos new file, a continuación, empty file > "Si" > save as: main.
Tendremos un archivo en nuestro proyecto: main.c.

Botones.

Botón amarillo - Build: Sirve para compilar.
Botón verde - Play: Ejecuta el programa
Botón con ambos símbolos - Build and play: Realiza ambas tareas anteriores, siempre que no tenga errores el código.
